Contrastes Ortogonales

Guía completa para entender y aplicar contrastes ortogonales en diseños experimentales

📚¿Qué son los Contrastes Ortogonales?

Un contraste es una combinación lineal de medias de tratamientos que permite hacer comparaciones específicas entre grupos. Son ortogonales cuando son independientes entre sí, proporcionando información única sobre diferentes aspectos de los datos.

Características principales:

  • Planificados: Se definen antes de recoger los datos
  • Específicos: Responden preguntas de investigación concretas
  • Independientes: Cada contraste aporta información única
  • Eficientes: Maximizan el uso de la información disponible

Fórmula general de un contraste:

$$C = c_1\mu_1 + c_2\mu_2 + ... + c_k\mu_k$$

donde: $$\sum_{i=1}^{k} c_i = 0$$

🧪Ejemplo Práctico: Abonos

Consideremos un experimento con tres tipos de abonos para evaluar su efecto en el crecimiento de plantas:

T₁: Humus

Abono Orgánico

Media: μ₁

T₂: Compost

Abono Orgánico

Media: μ₂

T₃: Urea

Abono Químico

Media: μ₃

Preguntas de investigación:

1. ¿Los abonos orgánicos como grupo son diferentes del químico?

2. ¿Hay diferencia entre los dos abonos orgánicos?

⚡Construcción de los Contrastes

1

Contraste 1

Orgánicos vs. Químico

$$C_1 = (\mu_1 + \mu_2) - 2\mu_3$$
$$C_1 = \mu_1 + \mu_2 - 2\mu_3$$

Coeficientes: [1, 1, -2]

2

Contraste 2

Entre Orgánicos

$$C_2 = \mu_1 - \mu_2$$

Coeficientes: [1, -1, 0]

Verificación de requisitos:

Suma de coeficientes = 0

C₁: 1 + 1 + (-2) = 0 ✓

C₂: 1 + (-1) + 0 = 0 ✓

Ortogonalidad (producto punto = 0)

(1×1) + (1×-1) + (-2×0) = 1 - 1 + 0 = 0 ✓

📊Tabla de Coeficientes

Contraste T₁ (Humus) T₂ (Compost) T₃ (Urea) Interpretación
C₁ +1 +1 -2 Orgánicos vs. Químico
C₂ +1 -1 0 Humus vs. Compost

Ejemplo con datos:

Supongamos: μ₁ = 25, μ₂ = 23, μ₃ = 18

C₁ = 25 + 23 - 2(18) = 48 - 36 = 12

C₂ = 25 - 23 = 2

Interpretación:

  • C₁ > 0: Los orgánicos superan al químico
  • C₂ > 0: El humus supera al compost

🎯Pasos para Aplicar Contrastes Ortogonales

1

Planificación

Define las comparaciones de interés antes de recoger datos

2

Construcción

Asigna coeficientes que sumen cero para cada contraste

3

Verificación

Confirma que los contrastes son ortogonales

4

Análisis

Calcula estadísticos de prueba e interpreta resultados

💡Ventajas y Consideraciones

✅ Ventajas

  • Mayor poder estadístico
  • Control del error tipo I
  • Respuestas específicas
  • Interpretación clara

⚠️ Consideraciones

  • Planificación previa obligatoria
  • Máximo k-1 contrastes
  • Deben ser ortogonales
  • Interpretación contextual

Conceptos Relacionados:

ANOVA Comparaciones Múltiples Diseño Experimental Suma de Cuadrados

Suma de cuadrados para contraste:

$$SS_C = \frac{(\sum c_i \bar{y}_i)^2}{r\sum c_i^2}$$

Estadístico F:

$$F = \frac{MS_C}{MS_E} = \frac{SS_C}{MS_E}$$

donde: r = repeticiones, MS_E = cuadrado medio del error
